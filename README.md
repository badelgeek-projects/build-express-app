> Ready to go app folder with different folder structure
- basic
- middleware
- ... (coming soon)

## Usage

### Default options
```sh
npx build-express-app appName
```
### Folder Structure
```sh
npx build-express-app --folder middleware appName
```

### Listening Port
```sh
npx build-express-app --port 8080 appName
```


## Help

```sh
npx build-express-app help
```
